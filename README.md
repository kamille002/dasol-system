## 🚀 실행 방법

### 1단계: index.html 더블클릭

### 2단계: 로그인
```
이메일: admin@dasol.co.kr
비밀번호: admin1234
```
(이미 자동 입력되어 있습니다!)

### 3단계: 대시보드 확인
- 📢 회사 공지사항 / 관보 업데이트
- 💛 출근 파트
- 💚 수질자가측정 파트
- 💙 먹는물 수질검사 파트
- 💗 영업팀 전용 모드
- **🧡 내무행정 파트** ⭐ 여기에 전자결재!

### 4단계: 전자결재 사용
1. 내무행정 파트의 **"📋 전자결재"** 클릭
2. 10종 문서 목록 팝업
3. 원하는 문서 선택
4. 작성 후 결재 요청!

## 💡 핵심 기능

### 1️⃣ 금액별 자동 결재선 (지출결의서)
```
10만원 미만    → 팀장
100만원 미만   → 팀장 → 부장
100만원 이상   → 팀장 → 부장 → 실장 → 대표
```

### 2️⃣ 근태 유형 통합 (근태신청서)
```
🏖️ 연차
⏰ 반차 (오전/오후)
🏥 병가
💐 경조사
🚶 외출
🏃 조퇴
```

### 3️⃣ 품의서 (만능 결재)
- 위 문서로 분류 안되는 모든 결재
- "유류대 인상", "비품 구매", "거래처 접대" 등
- 가장 많이 쓰이는 양식
